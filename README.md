# SingleNet
闪讯安卓客户端、小部件。
## 支持
+ 安卓5.1以上，支持金立F103S。
## 功能
+ 密码过期时，发送短信mm至闪讯号获取新密码。
+ 小部件显示密码、已过期或者无数据。
+ 双卡手机支持。
